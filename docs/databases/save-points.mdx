# Save Point

Save point is a SQL feature that can help us group queries **within a transaction** before doing a commit.
Transactions provide atomicity.
But within a large transaction, save points provides a pointer t

:::important rollback is only before a commit
ROLLBACK is a feature that can be used only before a commit.

If a commit is done, then we can only use DELETE statements to remove changes.
:::
