# Topological Sort

Topological sort is about sorting the nodes in a graph based on its dependencies.

:::info Mental model for the name
Topology is what we come across when discussing about the network setup.
It means, the logical structure of the network.

Same applies here, based on the structure of the graph, we sort the nodes in a topological sort.
:::

:::danger Not a regular sort
Topological sort doesn't sort the elements based on its value like we do it in
Insertion Sort, Bubble Sort, Merge Sort.
:::

Topological sort needs two important components

1. **Adjacency list** - List of outgoing connections from a node.
2. **In-degree** - the number of incoming connections for a node.

:::warning Contents of Adjacency list and In-degree
Adjacency list contains the actual node names whereas In-degree contains only the count.
:::
