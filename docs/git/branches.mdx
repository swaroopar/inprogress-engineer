# Branches

Every branch we create locally has no relation to the branch on the remote at all.
Only there is one small reference in the metadata,
which says that a specific local branch is linked to a specific remote branch.

They're two **parallel** branches, which aren't connected in any way.

:::tip remote to local relation
All relation to GIT local to remote branches are stored in the metadata of the local repository.
It's defined in the `.git/config` file.
:::
